功能

我们可以做什么

01.

文书分析

可获取一段时间内的文书。上传某一文书可获得摘要、基本信息、自动分词并生成标注，还有相似案件的智能推荐。
02.

案例库

了解典型案例的文书全文、摘要、关键词等信息。
03.

数据展示

根据文书信息统计结果，绘制而成的可视化图表。
04.

法条推荐

进行法条分类、高频引用条文和依据的科普。
